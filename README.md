6. Seleccionar el intérprete o kernel de Python.
7. Ejecutar las celdas en orden.

## Qué hace el programa

El programa simula una competencia de cocina con 5 participantes y 5 rondas.

En cada ronda:
- se calcula el puntaje total de cada participante sumando los puntajes de los 3 jueces,
- se determina el ganador de la ronda,
- se actualiza el puntaje acumulado,
- se contabilizan las rondas ganadas,
- se guarda el mejor puntaje obtenido en una ronda,
- se calcula el promedio por ronda,
- y se imprime la tabla de posiciones correspondiente.

Al finalizar, se muestra una tabla final ordenada de mayor a menor según el puntaje total acumulado.
